# Responsive Testing

The skill automatically detects responsive behaviors and generates mobile-oriented E2E scenarios.

Supported:
- mobile navigation
- drawers
- overlays
- responsive layouts
- adaptive rendering
- breakpoint-based UI

---

# Devices

The skill uses:
- official Playwright devices
- custom viewport configurations

Supported breakpoints:
- mobile
- tablet
- desktop

---

# Main Goal

Validate that responsive interfaces behave correctly across different viewport sizes and interaction models.
